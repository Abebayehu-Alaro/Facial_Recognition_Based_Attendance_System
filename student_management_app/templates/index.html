    S.A.M.S is Facial Recognition Based Student Attendance Management System which aims to minimize paperwork and save time by automating the traditional way of taking attendance. It tracks the attendance of the students using Facial Recognition Model. It provides an user friendly interface for the users and enable them to perform their desired attendance related tasks according to their role. It is developed using Django Python Framework and Deep Learning Models.

FEATURE LIST

Facial Recognition based Student Attendance System has the following features.

Data Storage

With a single sign-on, faculties can access their own self-service accounts, along with a lot of other data in one place.

Automated Reports

Quite often teachers have to access the attendance reports of their students on an urgent basis. Using attendance monitoring system you basically make sure that reports can be created with just a click of your button.

Fast & Effective

Adopting automated attendance monitoring system will help save time and money by eliminating a great deal of manual processes involved in attendance and leave entry and calculating hours attended.
